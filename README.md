# 🚀 Pure Finance - Next-Gen Expense Manager

> **Pure Finance** is not just another weekend project. It’s a production-ready, fully localized, and secure Progressive Web Application (PWA) built with a strict product-first mindset. It helps users track expenses, manage budgets, and analyze financial habits with a flawless UX.

---

## ✨ Why is this different? (Unique Selling Proposition)

Instead of relying on annoying confirmation pop-ups, this app implements a **frictionless UX** inspired by top-tier products:
* 🔄 **Smooth Undo Action:** Accidentally deleted an expense? You have a 4-second Toast window to undo the action. No friction, just flow.
* 🔒 **Privacy First:** Built on top of **Supabase Row Level Security (RLS)**. Every user's data is cryptographically isolated at the database level.
* 📱 **Cross-Platform PWA:** Installable directly on iOS and Android devices without going through the App Store.

## 🧰 Tech Stack & Architecture

**Frontend:**
* **Core:** React 18 + Vite 5
* **Styling:** Tailwind CSS v3 (with Deep Dark Mode support)
* **State Management:** Zustand (lightweight & fast)
* **Routing:** React Router v6
* **Data Visualization:** Recharts (Line, Pie, Bar charts)
* **Localization:** i18next (PL / EN / DE support out of the box)

**Backend & Infrastructure:**
* **BaaS:** Supabase (PostgreSQL, Authentication, RLS)
* **Deployment:** Vercel

## 🚀 Key Features Include:

- **Comprehensive Dashboard:** Visual summaries of your spending and remaining budget.
- **Advanced Filtering:** Sort and search expenses by date, category, or amount.
- **Smart Budgets:** Set global limits and track progress with dynamic color-coded indicators (Green/Yellow/Red).
- **Customization:** Users can create, edit, and manage their own custom categories.
- **Internationalization:** Seamless language switching (English, Polish, German).
- **Empty States & Animations:** Thoughtful UI details when there is no data to display.
